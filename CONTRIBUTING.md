# Contributing

Thank you for helping improve Abyssal Planet.

## Bug Reports

Use the bug report template and provide clear reproduction steps, versions and logs. Test with the minimum required dependencies whenever possible.

## Suggestions

Feature ideas are welcome through the feature request template. Please explain how the feature fits the underwater exploration and progression of the mod.

## Code and Assets

Abyssal Planet is currently an All Rights Reserved project. Do not submit or redistribute modified builds without prior authorization. Contact the project owner before preparing a substantial code or asset contribution.
