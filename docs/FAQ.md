# Frequently Asked Questions

## Which Minecraft version is supported?

Minecraft Java Edition 1.20.1 with Forge 47.x.

## Which Java version is required?

Java 17.

## Is GeckoLib required?

Yes. GeckoLib 4.8.4 or newer for Forge 1.20.1 is a required dependency.

## Does the mod work on servers?

Yes. Install the same Abyssal Planet and GeckoLib versions on the server and every connecting client.

## Can the mod be added to an existing world?

Back up the world first. New dimension terrain improvements generally appear in newly generated chunks.

## Can the mod be included in a modpack?

Contact the project owner for permission when distribution is not handled directly through the official CurseForge project.

## Where should bugs be reported?

Use the GitHub Issues tab and include logs, versions and reproduction steps.
